Vuejs의 Computed

Computed는 Vue.js에서 사용되는 컴포넌트의 Property입니다.

export default {
  computed: {
    sum() { // 3
      const { a, b } = this;
      return a + b;
    },
  },
  data() {
    return {
      a: 1,
      b: 2,
    };
  },
}

위 코드와 같이 computed Property 내부에 로직을 정의하면 내부에서 사용 된 상태값이 변경 될 때마다 해당 상태값도 같이 변경되어 반환되는 기능을 하고 있습니다.
